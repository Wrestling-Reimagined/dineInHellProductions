### Scripts

- **Install dependencies**

```bash
npm install
```

- **Start dev server**

```bash
npm run dev
```

- **Build for production**

```bash
npm run build
```

- **Preview production build**

```bash
npm run preview
```

### Tech stack

- **React 18**
- **TypeScript**
- **Vite** (with `@vitejs/plugin-react-swc`)

Main entry is in `src/main.tsx`, and the root component is in `src/App.tsx`.
